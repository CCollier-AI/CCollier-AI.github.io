# Building with AI

This repository hosts my personal writing and notes on building with AI,
alongside an MBA I’m starting in April 2026.

The focus is practical and reflective rather than hype-driven:
- how AI supports (or hinders) real thinking
- decision-making and trade-offs
- product and system design
- lessons from building and experimenting

The site is published via GitHub Pages and lives here:
https://CCollier-AI.github.io/

- A public notebook of learning and experimentation
- Long-form writing rather than social media posts
- Personal work, not employer-sponsored

## What this is not

- A company blog
- Documentation for proprietary systems
- A collection of prompts or tutorials
